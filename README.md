# espacial-funkopop
Protótipo de portfólio da minha loja pessoal "Espacial Funko Pop", focado na apresentação de Funko Pops personalizados feitos em biscuit. O projeto tem como objetivo divulgar os produtos, explorar o design da marca e servir como base para futuras melhorias e funcionalidades.
